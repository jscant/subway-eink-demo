Demonstration of ESP32 wireless subway board for home use

This project showcases a custom-built ESP32-powered e‑ink subway display designed for home use. The device connects over Wi‑Fi, retrieves real‑time subway arrival data from the Seoul Open Data API, and displays the upcoming train information using an ultra‑low‑power e‑ink panel.

The goal of this demonstration is to show how open public transit data can be integrated with affordable microcontrollers to create practical, always‑on home information devices.

About the System

The system performs periodic API calls to retrieve the latest train arrival times and updates the e‑ink display accordingly. Because e‑ink only uses power when refreshing, the overall device operates with very low energy consumption, making it ideal for continuous home use.

This webpage is provided as part of a usage‑case submission demonstrating integration with the Seoul Open Data subway arrival API.
